# Руководство по размещению на хостинге

## Рекомендуемый хостинг: InfinityFree

### Преимущества:
- ✅ Полностью бесплатный
- ✅ PHP 7.4-8.2 + MySQL
- ✅ Без рекламы
- ✅ Бесплатный SSL сертификат
- ✅ 5 ГБ места, неограниченный трафик
- ✅ Подходит для студенческих проектов

### Регистрация и размещение:

1. **Регистрация:**
   - Перейдите на https://infinityfree.net
   - Нажмите "Sign Up" (Регистрация)
   - Заполните форму регистрации

2. **Создание сайта:**
   - Войдите в панель управления
   - Нажмите "Create Account" (Создать аккаунт)
   - Выберите домен (можно использовать бесплатный поддомен типа `yourname.infinityfreeapp.com`)
   - Заполните форму

3. **Загрузка файлов:**
   - В панели управления нажмите "File Manager" (Файловый менеджер)
   - Загрузите все файлы проекта в папку `htdocs` или `public_html`
   - **ВАЖНО:** Загрузите файлы в правильном порядке:
     1. Сначала все PHP файлы
     2. CSS и JS файлы
     3. Папку images (если есть)

4. **Создание базы данных:**
   - В панели управления перейдите в "MySQL Databases"
   - Создайте новую базу данных
   - Запомните:
     - Имя базы данных
     - Имя пользователя
     - Пароль
     - Хост (обычно `sqlXXX.infinityfree.com`)

5. **Настройка config.php:**
   ```php
   define('DB_HOST', 'sqlXXX.infinityfree.com'); // Хост из панели
   define('DB_USER', 'epiz_XXXXXX'); // Имя пользователя БД
   define('DB_PASS', 'ваш_пароль'); // Пароль БД
   define('DB_NAME', 'epiz_XXXXXX_dbname'); // Имя базы данных
   ```

6. **Импорт базы данных:**
   - В панели управления перейдите в "phpMyAdmin"
   - Выберите вашу базу данных
   - Нажмите "Import" (Импорт)
   - Выберите файл `database.sql`
   - Или используйте `setup.php` через браузер

7. **Проверка:**
   - Откройте ваш сайт в браузере
   - Проверьте работу всех страниц
   - Протестируйте регистрацию и вход

## Альтернативные варианты

### 1. Локальный сервер + ngrok (для демонстрации)

Если нужно быстро показать проект преподавателю:

1. **Установите XAMPP:**
   - Скачайте с https://www.apachefriends.org
   - Установите и запустите Apache + MySQL

2. **Настройте проект:**
   - Скопируйте файлы в `C:\xampp\htdocs\РК2`
   - Создайте БД через phpMyAdmin
   - Настройте `config.php`

3. **Используйте ngrok для внешнего доступа:**
   ```bash
   # Установите ngrok с https://ngrok.com
   ngrok http 80
   ```
   - Вы получите публичную ссылку типа `https://xxxx.ngrok.io`

### 2. 000webhost (альтернатива)

- Сайт: https://www.000webhost.com
- Регистрация похожа на InfinityFree
- Менее стабильный, но работает

### 3. Российские хостинги (бесплатные тесты)

- **Beget:** https://beget.com (30 дней бесплатно)
- **Timeweb:** https://timeweb.com (тестовый период)

## Важные моменты

### Безопасность:
1. После размещения удалите `setup.php` или ограничьте к нему доступ
2. Проверьте права доступа к файлам (обычно 644 для файлов, 755 для папок)
3. Не храните пароли БД в открытом виде (используйте переменные окружения, если возможно)

### Изображения:
- Файлы изображений в БД ссылаются на `images/`
- Убедитесь, что папка `images` создана и доступна для записи
- Или загрузите изображения на внешний хостинг (imgur, imgbb) и обновите ссылки в БД

### Проверка работы:
После размещения проверьте:
- [ ] Главная страница открывается
- [ ] Навигация работает
- [ ] Страница магазина отображает товары
- [ ] Регистрация и вход работают
- [ ] Корзина функционирует (после входа)
- [ ] Форма обратной связи отправляется

## Решение проблем

### Ошибка подключения к БД:
- Проверьте данные в `config.php`
- Убедитесь, что БД создана
- Проверьте права пользователя БД

### Файлы не загружаются:
- Проверьте права доступа
- Убедитесь, что файлы в правильной папке (htdocs/public_html)

### Сессии не работают:
- Проверьте настройки PHP для сессий
- Убедитесь, что `session_start()` вызывается

### Изображения не отображаются:
- Проверьте пути к изображениям
- Убедитесь, что папка images существует
- Проверьте права доступа
